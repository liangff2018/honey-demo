angular常用指令：
1、创建directive指令：ng g directive button
2、启用服务：ng serve
3、启动服务并直接在浏览器中打开：ng serve --open
4、指定端口启动服务并在浏览器中打开：ng serve --port 4201 --open 
5、创建组件：ng generate component heroes //使用 Angular CLI 创建一个名为 heroes 的新组件。
6、缩写方式创建组件：ng g component heroes
7、指定目录创建组件：ng g component demo/heroes
8、创建组件时不生成.spec.ts的测试文件：ng g c --spec=false heroes
9、创建服务：ng generate service hero
10、打包：ng build -prod，  缩写：ng b -prod
